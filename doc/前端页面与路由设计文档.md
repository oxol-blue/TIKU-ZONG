# 前端页面与路由设计

## 1. 技术与目录

前端基于 koi-ui、Vue 3、TypeScript、Vite、Pinia 和 Element Plus，位于 `frontend/src/`。题库页面位于 `views/tiku/`，接口集中在 `api/tiku/index.ts`。

## 2. 用户端

```text
/tiku/search      在线搜题、相似度、AI 标识和答案反馈
/tiku/packages    套餐购买、优惠券和我的套餐
/tiku/account     API Key 和 OCS 配置生成
```

## 3. 管理端

`/tiku/admin` 使用页签承载 OCS 源、AI 模型、AI 答案、支付、套餐/优惠券、用户、题库、订单和调用日志管理。管理员 TOTP 保存在当前浏览器 `sessionStorage`，不写入后端或 URL。

## 4. 约定

接口统一通过 `koi` Axios 封装发送，页面只使用后端返回的文字答案；多选答案展示为 `###` 分隔。所有新页面应补充加载态、空态、错误提示和移动端布局。
