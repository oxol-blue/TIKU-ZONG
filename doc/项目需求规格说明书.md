# 题库调用系统需求规格说明书

## 1. 项目范围

面向学生、教师等网课用户，预计支持 100～500 人。系统为单租户、中文界面，提供在线搜题、API 调用、API Key 鉴权、OCS AnswererWrapper 兼容和 AI 兜底。

## 2. 角色与账号

系统包含管理员和用户两种角色。用户使用邮箱和密码登录，每个用户最多一个 API Key；数据库只保存 API Key 哈希。管理员负责题库、套餐、订单、支付、AI、OCS、用户和日志管理。

## 3. 核心规则

支持选择、多选、判断、简答、填空及扩展题型。答案保存和返回选项文字，多选答案使用 `###`。题目按标准化题目、选项和答案计算 SHA-256 去重，并提供相似题召回。

套餐包括时间、次数、时间次数三类，允许叠加；普通调用和 AI 调用分别计费。题库、OCS 无答案时调用 AI，失败不扣费，AI 结果保存到 `question_ai`。

## 4. 非功能要求

MySQL 5.7、Go/Gin、Vue/koi-ui 前后端分离；生产环境需要 HTTPS、密钥隔离、登录保护、图形验证码、IP 访问控制、API 防刷、管理员 TOTP、备份和审计。
