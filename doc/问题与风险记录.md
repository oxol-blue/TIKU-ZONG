# 问题与风险记录

## 当前风险

- 相似度为轻量字符二元组算法，尚未用大规模真实题库做阈值和误召回评估。
- Redis 分布式限流已实现，但尚未在生产 Redis 上完成连接、ACL 和故障回退演练。
- 易支付退款目前记录内部退款状态，真实渠道退款调用和对账仍需补齐。
- 订单自动维护依赖服务进程持续运行，生产应增加进程守护和告警。
- 监控模板已提供，但仍需在实际 Prometheus/告警接收器中验证抓取、通知路由和告警恢复。
- 管理统计当前为实时聚合查询，大数据量下应增加按日汇总或只读统计库，避免影响业务数据库。
- AI 原始响应可能包含敏感题目内容，数据库和备份必须严格控制访问。
- 测试管理员账号仅用于本地/测试环境，生产环境必须使用独立强密码并通过环境变量初始化，禁止复用测试凭据。

## 缓解措施

生产开启 HTTPS、TOTP、独立加密密钥和数据库最小权限；为备份和 AI 调用设置审计；上线前使用真实样本评估相似度，并保留人工停用 AI 缓存记录的能力。
