# 版本发布记录

## 当前版本

分支：`main`。版本以 Git 提交和 GitHub Actions CI 状态为准；每次合并前必须通过后端测试/Vet 与前端类型检查/构建。

## 已交付能力

认证与 API Key、题库导入/搜索/去重、套餐与支付、优惠券、邀请码、AI 兜底、OCS 兼容、安全控制，以及管理员用户、题库、订单、调用日志、AI 答案、公告、系统设置管理。已提供订单套餐实例补偿、支付结果页、GitHub CI、MySQL 备份恢复脚本和宝塔/Nginx/systemd 部署模板。
