# 功能补充记录：套餐编辑与优惠券状态

## 本次完成

- 新增管理员套餐编辑接口 `PUT /api/v1/admin/packages/{id}`。
- 编辑接口复用套餐类型、有效期、次数、AI 次数、价格和标记校验。
- 新增管理员优惠券启用/停用接口 `PATCH /api/v1/admin/coupons/{id}/status`。
- 管理后台套餐列表支持打开编辑弹窗并保存。
- 管理后台优惠券列表支持启用和停用。

## 验证

- `backend`: `go vet ./...` 通过。
- `backend`: `go test ./...` 通过。
- `frontend`: `pnpm run type:check` 通过。
- `frontend`: `pnpm run build` 通过。

## 业务说明

- 下架套餐不会删除已有用户套餐实例，只影响后续购买。
- 停用优惠券不会修改已创建订单，只阻止后续订单继续使用。
- 编辑套餐会影响后续发放的套餐实例，已有实例保留创建时的有效期和余额。
