# 功能补充记录：系统配置

## 管理能力

新增系统配置表和管理员页面，用于维护：

- 站点名称；
- 支持链接；
- 维护公告；
- 新用户注册开关。

公开接口 `GET /api/v1/settings/public` 返回站点名称、支持链接、维护公告和注册开关。管理员使用 `GET`、`PUT /api/v1/admin/settings` 读取和修改完整配置；登录页据此隐藏已关闭的注册入口，后端仍执行强制校验。

## 安全和行为

配置模块不保存密码、API Key、AI 密钥或支付凭据。注册开关在注册路由前检查：关闭后返回 `REGISTRATION_CLOSED`；配置读取异常时返回 `503 SETTINGS_UNAVAILABLE`，避免意外开放注册。

## 迁移

`000015_system_settings.sql` 兼容 MySQL 5.7，并初始化默认配置。部署时执行 `go run ./cmd/migrate` 后重启后端服务。
