# 开发实施计划

## 已完成

基础认证、API Key、题库导入/去重/搜索、套餐计费、易支付基础闭环、优惠券、验证码、IP 控制、TOTP、OCS DSL、AI 兜底、管理端用户/题库/订单/日志/AI 页面已实现并通过现有测试。

## 近期工作

1. 用真实题库样本调优相似度阈值和选项权重。
2. 增加 Redis 分布式限流和队列化 AI 调用。
3. 完善易支付真实退款调用、退款对账和订单补偿任务。
4. 补充邀请码、用户订单历史和正式全链路验收。
5. 完成宝塔部署、备份恢复演练和监控告警。

## 交付节奏

每个迭代按功能拆分 Git 提交，先补测试，再更新《开发过程记录》和《变更记录》。发布前运行 `go test ./...`、`go vet ./...`、`pnpm run type:check` 和 `pnpm run build`。
